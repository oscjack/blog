Ubuntu 12.04中安装Sublime Text2

作者: 360weboy

QQ群: 197642724

Sublime Text2这个编辑器据说非常好用，今天在http://www.sublimetext.com上下载了一个包，安装了一下。 具体安装步骤如下：

第一步

用以下命令将下载的包解压:

    
        tar xf Sublime\ Text\ 2.0.1\ x64.tar.bz2

我下载的是针对linux 64位的包，因为64位的运行速度相对快些。推荐你安装64位的。

注意上述\是用来转义空格的。

第二步

解压后，你会看到Sublime Text2这个目录，个人觉得应该把它放到根目录下的opt中比较合适，所以我运行下边的命令：

    
        sudo mv Sublime\ Text\ 2 /opt/

第三步

为了能在终端中可以直接通过输入sublime来打开所需文件，我建立了一个到sublime的软链接:

    
        sudo ln -s /opt/Sublime\ Text\ 2/sublime_text /usr/bin/sublime

第四步

通过建立一个sublime.desktop文件，将sublime加入到应用程序菜单中。

    
        sudo sublime /usr/share/applications/sublime.desktop

拷贝下属内容到改文件中：

    
        [Desktop Entry]
        Version=1.0
        Name=Sublime Text 2
        # Only KDE 4 seems to use GenericName, so we reuse the KDE strings.
        # From Ubuntu's language-pack-kde-XX-base packages, version 9.04-20090413.
        GenericName=Text Editor
        
        Exec=sublime
        Terminal=false
        Icon=/opt/Sublime Text 2/Icon/48x48/sublime_text.png
        Type=Application
        Categories=TextEditor;IDE;Development
        X-Ayatana-Desktop-Shortcuts=NewWindow
        
        [NewWindow Shortcut Group]
        Name=New Window
        Exec=sublime -n
        TargetEnvironment=Unity

第五步

设置默认用sublime打开的文档类型，运行下面的命令行打开默认关联的文件列表：

    
        sudo sublime /usr/share/applications/defaults.list

搜索gedit.desktop，然后将它们替换为sublime.desktop

安装结束！你可以使用sublime做为日常文件编辑器了，但是个人开发项目时，还是比较喜欢使用eclipse, 不过有时用用sublime处理一些文件，感觉也很方便。建议你试用下！
